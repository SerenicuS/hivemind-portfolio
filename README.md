# HiveMind Portfolio

A terminal-based portfolio built with Rust and Dioxus. Type commands to explore my work and background.

## Quick Start

```bash
git clone https://github.com/SerenicuS/hivemind-portfolio
cd hivemind-portfolio
cargo install dioxus-cli
dx serve
```

Visit `http://localhost:8080`

## Commands

- `help` - See all commands
- `whoami` - About me
- `skills` - My skills
- `projects` - My projects
- `socials` - Links to GitHub, LinkedIn, Facebook
- `showcase` - Demo gif
- `soul` - My poems
- `cv` - CV status
- `clear` - Clear screen
